# Rascunho simples para LinkedIn

Desenvolvi um projeto inicial de portfolio em Engenharia de Dados usando um dataset publico de FIIs.

Neste estudo, escolhi 4 tabelas do dataset para praticar etapas basicas do dia a dia:

- leitura de dados
- limpeza simples
- padronizacao de colunas
- join entre tabelas
- geracao de arquivos finais em CSV

No final, montei uma base consolidada e criei alguns outputs simples para analise.

Esse projeto faz parte do meu processo de aprendizado na area de dados, com foco em ganhar familiaridade com manipulacao e organizacao de datasets reais.
